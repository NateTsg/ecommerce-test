# Ecommerce API

This is the API for Ecommerce Platform

## How to Setup

### Installing Sequelize CLI

```bash
npm install --save sequelize-cli
```

### Bootstrapping Sequelize

```bash
npx sequelize-cli init
```

### Running Migrations 

```bash
npx sequelize-cli db:migrate
```

### Running Seeds 

```bash
npx sequelize-cli db:seed:all
```
